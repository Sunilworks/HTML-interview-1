1) Difference between div and span.

Div : Its bascically a block level element which is used to bind all the things together (seperate things) and
since it's a block level element it occupies the full width of the screen.

Span : Its an inline level element and it occupies only the required width of the screen.


2) Difference between class and IDs.

Class : Class selector can be applied to several elements on a single page.

IDs : ID selector is applied only to one element in a page.


3) What are self closing tags ?
Self closing tags are basically the tags which does not require to be closed manually.


4) How many tags of headings are there ? how to decide which to use where ?
There are basically 6 HTML Heading tags and the biggest is H1 and then H2 and H3 and so on.
Deciding which one to use is based on the user's preference.


5) Difference block and inline.
Block : It occupies full width of the screen.
Inline : It occupies only the required width of the screen.


6) Difference em and strong.
em : It makes the code in ITALICS and is considered as latest and good coding and also helps in SEO.
strong : It makes the code in BOLD and is considered as latest and good coding and also helps in SEO.


7) What are attributes ?
Attributes are basically which we use inside tags and it provides a special meaning,
provides additional information about elements.


8) Difference between HTML tags and elements.
HTML Tags : It is written between <(opening tag) and>(closing tag) and all the HTML tags are different.
Elements : It has opening tag + content + closing tag, the combination of all three things is called HTML element.
